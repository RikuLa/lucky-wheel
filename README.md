## Usage

Used APIs (**From roulette**)

- **Page Visibility API**
- **ReadableStream**
- **Resize Observer API**
- Service Worker API
- Clipboard API
- Web Audio API
- Web Storage API
- Canvas API

### Development

To start the app with hot module reloading run:

```
yarn start
```

### Chrome allow pop-ups

1. Settings
2. Site-settings
3. Pop-ups and redirects
4. Allow `localhost:1234`

### Production

To build the app in production mode run:

```
yarn build
```
